# Contributing

Thank you for your interest in contributing to Decision Passport OpenClaw Lite.

## What we welcome

- Bug reports with reproducible steps
- New example integrations (follow the pattern in `/examples/`)
- Documentation improvements
- Additional tool intent mappers and result summarisers

## How to contribute

1. Fork the repository
2. `git checkout -b feat/my-improvement`
3. `pnpm install --frozen-lockfile && pnpm test`
4. Open a Pull Request with a clear title and description

## Code standards

- TypeScript strict mode
- No external runtime dependencies in `src/`
- All exported classes/functions must have JSDoc
