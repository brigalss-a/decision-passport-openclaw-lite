# Decision Passport Lite for OpenClaw

**Description**: Add verifiable audit trails to OpenClaw in minutes.

## What it does
- records reasoning summaries
- records tool intent
- records result summaries
- exports a basic passport bundle

## What it does NOT do
- no private execution guard
- no claims engine
- no replay protection
- no enterprise control plane

## Install
```bash
clawhub install decision-passport-lite
```
